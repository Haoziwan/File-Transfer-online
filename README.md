An online file transfer application based on WebRTC that runs entirely in the browser without server storage.

##  Features

- **End-to-end Encryption** - Files transferred directly between devices without server intermediation
- **QR Code Sharing** - Scan QR code to receive files on another device
- **Real-time Progress** - Shows transfer speed and progress
- **Modern UI** - Dark theme with smooth animations
- **Large File Support** - Chunked transfer for files of any size


## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling framework
- **PeerJS** - WebRTC wrapper library
- **QRCode** - QR code generation

## Quick Start

### Install Dependencies

```bash
npm install
```

### Development Mode

```bash
npm run dev
```
